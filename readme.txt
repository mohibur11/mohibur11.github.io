Deltek Microsoft Outlook Add-Ins

Applicable for OutLook 2016 or later.

## Run as a user

Go to Deltek Add-Ins folder and double click on run.bat file.
It will open a node command prompt. Don't close it while using the Add-Ins.
And also it will automatically open the microsoft outlook application.

Now just use the Add-Ins.



For Developer:
Setup: 
1. Install the latest version of node.
2. Need a github account username which have at least two gists.(you can use username: "mohibur11".There have two gists).

Process:
1. Go to the file path and open a command window.
2. Run "npm start".
3. If anything goes wrong then run "npm install" and then run "npm start".


For Debugging in Visual Studio Code.
1. Open the project in Visual Studio Code.
2. Press Ctrl + Shift + D, a sidePane will open from left.
3. Choose "Outlook Desktop(Edge Chromium)" options from the dropdown.
4. Click on the Green Run Icon.
5. Debugging mode will be started.
